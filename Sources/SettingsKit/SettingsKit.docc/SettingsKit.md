# ``SettingsKit``

A lightweight Swift framework for creating macOS preferences windows with SwiftUI content.

## Overview

SettingsKit provides a simple, protocol-based API for building native macOS preferences windows. It bridges SwiftUI content to AppKit, allowing you to create modern settings interfaces with minimal boilerplate.

### Features

- Native macOS toolbar-style tab interface
- SwiftUI content support with AppKit integration
- Automatic window resizing when switching tabs
- Accessibility support (respects Reduce Motion setting)
